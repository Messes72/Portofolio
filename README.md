# Portfolio Website

Portfolio website modern dan interaktif untuk menampilkan project-project development.

## Tech Stack

- **Framework**: Next.js 15 dengan App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Animation**: Framer Motion + GSAP dengan ScrollTrigger
- **Icons**: Lucide React
- **Deployment**: Vercel

## Features

- Hero Section dengan animated text reveal dan floating shapes
- About Section dengan skills visualization
- Skills Section dengan animated bars
- Projects Gallery dengan magnetic cursor effects
- Project Detail Pages dengan documentation
- Contact Section
- Dark/Light mode toggle
- Smooth scrolling
- SEO optimized

## Development

### Prerequisites

- Node.js 18+
- npm/pnpm/yarn

### Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── app/              # Next.js app router pages
├── components/       # React components
├── lib/             # Utilities and data
├── types/           # TypeScript interfaces
└── public/          # Static assets
```
